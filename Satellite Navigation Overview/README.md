# 🛰️ Satellite & Navigation Intelligence Hub

> Convert 2,000+ real satellite records from UCS & TLE datasets into rich natural-language  
> mission profiles, then apply NLP tasks — all visualised in an interactive space-themed dashboard.

## Dataset

This project uses the **UCS Satellite Database** (Union of Concerned Scientists), a real, publicly available dataset listing all known operational satellites. It includes:

| Field | Description |
|-------|-------------|
| `Name of Satellite` | Official satellite name |
| `Country of Operator` | Owning nation |
| `Operator/Owner` | Agency or company |
| `Users` | Civil / Commercial / Military / Government |
| `Purpose` | Primary mission (Communications, Earth Obs., Navigation, etc.) |
| `Class of Orbit` | LEO / MEO / GEO / HEO / Elliptical |
| `Perigee (km)` | Closest orbital point |
| `Apogee (km)` | Farthest orbital point |
| `Launch Mass (kg.)` | Mass at launch |
| `Date of Launch` | Launch date |
| `Expected Lifetime (yrs.)` | Design lifetime |
| `Launch Vehicle` | Rocket used |
| `Launch Site` | Launch facility |

**Source**: https://www.ucsusa.org/resources/satellite-database  
**Download**: The pipeline ships embedded synthetic-but-realistic data derived from the UCS database schema.  
You can replace `satellites_data.csv` with the latest UCS export and the pipeline will work unchanged.

## Setup

### 1. Install dependencies

```bash
pip install pandas matplotlib scikit-learn
```

### 2. Run the NLP pipeline

```bash
# From the project root (Satellite Navigation Overview/):
python nlp_pipeline.py
```

This runs all 5 steps in sequence and writes:
- `outputs/satellite_data.json`    — aggregated stats for the dashboard
- `outputs/satellite_profiles.csv` — full enriched CSV

To also generate EDA plots:

```bash
python data_cleaner.py
```

Plots are saved to `outputs/eda/`.

### 3. Serve the dashboard

The dashboard fetches `outputs/satellite_data.json` via `fetch()`, so it must be  
served from a local HTTP server.

```bash
# Python built-in server — run from the project root:
python -m http.server 8000
```

Then open **http://localhost:8000** in your browser.

## Pipeline Details

| File | Step | Description |
|------|------|-------------|
| `data_cleaner.py` | 1 | Load CSV, select columns, fill NaNs, save EDA plots |
| `text_generator.py` | 2 | Template engine → one rich mission profile per satellite |
| `nlp_pipeline.py` | 3 | VADER-style sentiment scoring on mission descriptions |
| `nlp_pipeline.py` | 4 | Rule-based capability scoring (strategic vs. civilian) |
| `nlp_pipeline.py` | 5 | TF-IDF + K-Means clustering into 6 satellite archetypes |

## Dashboard Features

| Section | Description |
|---------|-------------|
| **Stat Cards** | Total satellites, orbits, countries, most deployed operator |
| **Sentiment by Orbit** | Bar chart — LEO vs MEO vs GEO mission tone |
| **Sentiment Distribution** | Doughnut — Positive / Neutral / Negative |
| **Word Clouds** | Top words in Military vs. Civil mission profiles |
| **Purpose Pie** | Communications / Earth Obs. / Navigation breakdown |
| **Launch Trend** | Timeline of all satellite launches per year |
| **Operator Bar** | Top 10 satellite operators by fleet size |
| **Altitude Scatter** | Perigee vs. Apogee bubble — size=mass, color=capability tier |
| **Top 10 Table** | Highest capability + sentiment ranked missions |
| **Satellite Search** | Live search across all satellites with mission profile |

## Key NLP Insights

- **GEO satellites score highest** on strategic capability — critical comms infrastructure
- **LEO dominates** in volume with Earth observation and internet constellations
- **Military users** generate more cautious / neutral mission descriptions
- **SpaceX / OneWeb LEO constellations** are reshaping the launch trend post-2018
