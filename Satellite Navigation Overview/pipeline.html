🔄 Pipeline Architecture
satellites_data.csv  →  data_cleaner.py  →  text_generator.py  →  nlp_pipeline.py  →  satellite_data.json  →  Dashboard
The pipeline is fully self-contained — if no CSV is present, it generates 2,050 realistic synthetic satellite records based on the UCS Satellite Database schema, ensuring the dashboard always renders with meaningful orbital intelligence data.
📋 Pipeline Steps
01
⚙️ Data Loading & Cleaning
data_cleaner.py loads the UCS Satellite Database CSV. If the file is missing, it generates 2,050 synthetic rows with realistic orbital parameters (perigee, apogee, inclination, period) computed from Keplerian orbital mechanics. Missing numeric values are filled using per-orbit-class medians, ensuring no NaNs propagate downstream.

Produces 5 EDA plots saved to outputs/eda/.
pandas matplotlib Orbit Physics EDA Plots
02
✍️ Mission Profile Generation
text_generator.py converts each satellite row into a rich, human-readable mission profile using a deterministic template engine. Classification helpers translate numeric fields into descriptive language — orbit class, mass category, lifetime tier, purpose verb, and user context are all computed rule-based, requiring no LLM.

Navigation-relevant satellites receive additional context about GNSS contribution and augmentation system support.
Template Engine Rule-Based NLG 2,050 Profiles GNSS Context
03
💬 VADER Sentiment Scoring
A lightweight VADER-style lexicon scores each mission profile. Positive signals include: precise, reliable, accurate, navigation, next-generation, centimetre-level. Negative signals include: surveillance, reconnaissance, military, elliptical, expired.

Compound scores are normalised to [-1, +1] and labelled as Positive (>0.05), Neutral, or Negative (<−0.05).
VADER Lexicon Compound Score 3-Class Label No External API
04
⭐ Capability Tier Scoring
A 12-point rubric scores each satellite's strategic importance. Points are awarded for orbit class (GEO = 4, MEO = 3, LEO = 1), mission purpose criticality (Navigation = +3, Communications = +2), launch mass, and expected lifetime.

Tiers: Strategic (9–12 pts), Operational (6–8 pts), Support (3–5 pts), Experimental (0–2 pts).
12-Point Rubric 4 Capability Tiers Orbit + Purpose + Mass Rule-Based
05
🔮 TF-IDF + K-Means Clustering
Mission profiles are vectorised using TF-IDF (600 features, English stopwords removed) and clustered into 6 mission archetypes using K-Means.

Archetypes: LEO Earth Watchers, GEO Comms Backbone, Navigation Constellation, Military & Surveillance, Scientific Explorers, Technology Demonstrators.

sklearn gracefully degrades to "unclustered" if not installed.
TF-IDF Vectorizer K-Means (k=6) 6 Archetypes sklearn
06
📊 Dashboard JSON Export
The final step aggregates all computed metrics into outputs/satellite_data.json:

• Sentiment by orbit class (avg)
• Purpose, orbit, capability, user distributions
• Launch year trend (1990–2025)
• Top 10 operators by fleet size
• Top 10 highest-scored missions
• Word frequencies (positive/negative)
• Scatter plot sample (350 satellites)
• Full search index (all 2,050 satellites)
